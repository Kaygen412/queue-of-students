# Инструктаж к программе "Очередь студентов (FIFO)"

## Описание программы
Данная программа реализует очередь студентов по принципу **FIFO (First In — First Out)**.  
Это означает, что **первый добавленный студент будет первым обработан или удалён из очереди**.

Очередь реализована с помощью **связного списка**, где каждый объект `Student` содержит ссылку на следующего студента.

---

## Структура программы

### Класс `Student`
Класс представляет одного студента и содержит:
- `Name` — имя студента
- `Age` — возраст студента
- `Next` — ссылка на следующего студента в очереди

### Класс `StudentQueue`
Класс реализует очередь студентов и содержит:
- `head` — первый студент в очереди
- `tail` — последний студент в очереди

Методы:
- `Enqueue()` — добавляет студента в конец очереди
- `Print()` — выводит всех студентов в очереди

### Класс `Program`
Содержит метод `Main`, в котором реализовано меню управления программой.
